# Inter font asset

`InterVariable.ttf` is the upstream Inter 4.1 variable font from
<https://github.com/rsms/inter/tree/v4.1/docs/font-files>.

- SHA-256: `4989B125924991B90D05B2D16E0E388C48F7D5BB8B30539BBF9C755278D0CCAF`
- License: SIL Open Font License 1.1, stored beside the font as
  `Inter-OFL.txt`.

UAView Studio loads the staged copy at runtime and falls back to Dear ImGui's
built-in font only when the asset is unavailable.
